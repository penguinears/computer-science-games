Little Man Computer (LMC) Tutorial

The Little Man Computer (LMC) is a simple computer simulator used to learn the basics of assembly language and how a CPU works. It models a computer with 100 memory locations (numbered 00–99) and uses a small set of instructions to perform calculations, store data, and make decisions.

The LMC Interface

The simulator contains several main sections:

  • Code Editor – Where you write your LMC program.
  • Load Button – Compiles your program into memory.
  • Run Button – Executes the program.
  • Input Box – Used when the program asks for user input.
  • Output Window – Displays results produced by the program.
  • Memory – 100 memory locations (00–99) that store instructions and data.
  • Program Counter – Highlights the instruction currently being executed.
  • Speed Controls – Change how quickly the program runs.

LMC Instructions

Instruction Code Description
INP901Read a value from the user.
OUT902Display the value in the accumulator.
ADD1XXAdd a value from memory.
SUB2XXSubtract a value from memory.
STA3XXStore the accumulator in memory.
LDA5XXLoad a value from memory.
BRA6XXJump to another instruction.
BRZ7XXJump if the accumulator equals 0.
BRP8XXJump if the accumulator is positive or zero.
HLT000Stop the program.
DAT-Create a memory location for data.

Example 4 - Making Decisions

Branch instructions allow the program to choose different paths depending on a result.

  • BRA - Always jumps.
  • BRZ - Jumps only if the accumulator is zero.
  • BRP - Jumps if the accumulator is positive or zero.

The following program outputs the larger of two numbers.

INP
STA A
INP
STA B

LDA A
SUB B
BRP FIRST

LDA B
OUT
HLT

FIRST LDA A
OUT
HLT

A DAT
B DAT

Example 5 - Countdown Loop

Loops repeat instructions until a condition is met.

INP
STA COUNT

LOOP LDA COUNT
OUT
SUB ONE
STA COUNT
BRZ END
BRA LOOP

END LDA COUNT
OUT
HLT

COUNT DAT
ONE DAT 1

Example 6 - Multiplication

LMC has no multiply instruction, so multiplication is performed by repeatedly adding one number to itself.

Example: 3 × 4 = 3 + 3 + 3 + 3

Example 7 - Division

Division is achieved by repeatedly subtracting the divisor until the dividend reaches zero. The number of subtractions performed is the answer.

Example: 12 ÷ 3 

12 → 9 → 6 → 3 → 0

Answer = 4

Example 8 - Squaring a Number

To calculate a square, repeatedly add the number to itself the same number of times.

Example: 

5² = 5 + 5 + 5 + 5 + 5 = 25

Summary

The Little Man Computer is designed to teach the fundamentals of programming, including variables, memory, input/output, arithmetic, loops, and branching. Although it only supports a small number of instructions, it demonstrates how real processors execute programs one instruction at a time.
